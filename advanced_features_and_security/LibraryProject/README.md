# Managing Permissions and Groups in Django

This project demonstrates how to manage permissions and groups in Django.

## Features
- Custom model permissions (can_view, can_create, can_edit, can_delete)
- Permission enforcement using the permission_required decorator
- Group-based access control via Django admin

## Usage
Assign users to groups (Admins, Editors, Viewers) in the Django admin
and grant appropriate permissions to control access to views.
